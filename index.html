Morse Code Translator in C++



This is a Morse code encoder and decoder I created in C++. The encoding is done using a simple switch case, and the decoding is done based on a binary tree layed out in inorder order.

Check it out on GitHub

ZinckLib



This is a library I created for Python 3.10. The main focus of this library is around the manipulation of strings between various forms like strings to numbers or converting a number to its written form. There are also some other tools like creating a list of prime numbers less than a given value or determining if a string is in alphabetical order.

Check it out on GitHub
